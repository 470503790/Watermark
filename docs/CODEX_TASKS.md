# CODEX 任务卡（首版）

> 目标：在现有 M1 代码基础上，按里程碑逐步实现稳定高性能版本。每张卡含 DoD。

## M1-A 画布交互增强
- 实现八个缩放手柄 + 旋转把手的命中测试与拖拽
- Shift 等比缩放，Alt 以中心缩放
- DoD：操作平滑、无异常；变换矩阵正确应用到绘制与导出

## M1-B 文本样式扩展
- 字间距、行距、描边、阴影、文本框背景/圆角
- DoD：示例模板可验证所有参数可视化生效，导出一致

## M1-C 平铺水印（基础）
- 文本/图片层支持平铺：角度、间距、交错
- DoD：4K 图片上测试铺满且无缝

## M2-A 批处理与并发
- 输入文件夹扫描、命名模板、并发导出（有界）
- 原子写（.tmp→rename）、失败重试（2 次）、报告（CSV）
- DoD：1000 张 4K 稳定通过；失败率 <0.5%

## M2-B 模板包（ZIP）
- 打包 template.json + assets + fonts + manifest
- 加载包时校验缺失资产/字体，提供回退/阻止选项
- DoD：跨机器复现模板版式一致

## M3-A 数据源绑定（CSV/Excel/JSON）
- 字段映射向导、占位符与表达式求值（白名单函数）
- 可见性/位置/样式绑定
- DoD：5000 行数据驱动导出通过

## M3-B 二维码/条码函数
- 提供 `qr(text,size)`、`barcode(text,type,width,height)` 生成图片层
- DoD：示例模板内验证输出正确

## M3-C 瓦片渲染与大图优化
- 阈值启用 2048/4096 瓦片渲染，2px 重叠消除接缝
- DoD：>8K 巨图导出稳定，内存峰值显著下降
